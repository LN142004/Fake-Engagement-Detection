# Behavioural Analytics for Fake Engagement Detection in Social Media

## Problem Statement

Social media platforms suffer from fake engagement generated by automated bot accounts. These bots exhibit distinct behavioural patterns such as abnormal posting frequency, engagement spikes, and irregular activity timing.

This project builds a behavioural analytics model to detect fake engagement and classify accounts as Bot or Genuine.

---

## Dataset Information

Dataset Type: Hybrid (Public dataset + Behavioural Simulation)

Source: Provided hackathon dataset

Number of Records: 41,661

Each record contains:

* User ID
* Tweet
* Retweet Count
* Mention Count
* Follower Count
* Verified Status
* Bot Label
* Timestamp

---

## Why Behavioural Simulation was Required

The provided dataset contained limited behavioural indicators such as posting frequency and activity patterns.

Real-world bot detection depends heavily on behavioural features like:

* Posting frequency
* Activity timing
* Engagement ratio

Since these were not available, behavioural features were simulated using statistically realistic assumptions.

This is allowed under hackathon rules.

---

## Behavioural Features Generated

### Tweet Frequency

Simulated using normal distribution:

Bots:
Mean = 50 tweets/day
Std = 10

Humans:
Mean = 10 tweets/day
Std = 5

---

### Night Activity

Bots:
70% probability of night activity

Humans:
30% probability

---

### Engagement Ratio

Formula:

engagement_ratio = (retweets + mentions) / followers

Bots tend to show abnormal engagement ratios.

## Model Used

RandomForest Classifier

Reason:

* Works well for behavioural data
* Handles nonlinear relationships
* Provides feature importance for explanation

---

## Model Performance

Accuracy: 99.6%
Precision: 99.8%
Recall: 99.5%
F1 Score: 99.6%

---

## Behavioural Insights

Bots exhibit:

* Higher tweet frequency
* Increased night activity
* Abnormal engagement ratios

Tweet frequency and night activity were the strongest predictors.

---

## Anomaly Detection

Isolation Forest was used to detect coordinated behavioural anomalies.
