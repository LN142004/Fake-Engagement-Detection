# Behavioural Analytics for Fake Engagement Detection using XGBoost

## Problem Statement

Fake engagement on social media is often generated by automated bot accounts that exhibit abnormal behavioural patterns such as excessive posting frequency, abnormal engagement ratios, and irregular activity timing.

The objective of this project is to develop a behavioural analytics model that detects fake engagement and classifies accounts as Bot or Genuine using machine learning.

---

## Dataset Information

Dataset Type: Hybrid (Hackathon dataset + Behavioural Simulation)

Total Records: 41,661

Original Features:

* User ID
* Tweet
* Retweet Count
* Mention Count
* Follower Count
* Verified Status
* Bot Label
* Created At
* Hashtags

---

## Why Behavioural Simulation was Necessary

The provided dataset lacked key behavioural indicators required for behavioural analytics, such as:

* Posting frequency
* Activity timing patterns
* Behavioural engagement anomalies

Since behavioural analytics requires these features, additional behavioural indicators were simulated using statistical modelling based on known bot behaviour patterns.

This approach is allowed as per hackathon rules.

---

## Behavioural Features Engineered

### Engagement Feature

engagement = Retweet Count + Mention Count

Detects abnormal engagement bursts.

---

### Engagement Ratio

engagement_ratio = (Retweet Count + 1) / (Follower Count + 1)

Bots often exhibit abnormal engagement ratios.

---

### Posting Hour

Extracted from Created At timestamp.

Detects abnormal timing behaviour.

---

### Tweet Frequency (Simulated)

Bots:
Mean = 50 posts/day
Std Dev = 10

Humans:
Mean = 10 posts/day
Std Dev = 5

Simulated using normal distribution.

---

### Night Activity (Simulated)

Bots:
70% probability of night activity

Humans:
30% probability

---

## Model Used: XGBoost Classifier

XGBoost was selected because:

* High performance for classification tasks
* Handles nonlinear behavioural relationships
* Robust against overfitting
* Provides feature importance for behavioural explanation

---

## Behavioural Anomaly Detection

Isolation Forest was used to detect behavioural anomalies and coordinated bot activity patterns.

---

Accuracy: 99.68%

Precision: 99.85%

Recall: 99.51%

F1 Score: 99.68%
